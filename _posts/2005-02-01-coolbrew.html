Cooolbrew Framework/CMS

(Website is no longer available)

The project: Redesign the hcgPublic framework using a proper PHP framework as a base and build a CMS as part of it.

The goal with this new framework was to create the next generation of hcgPublic. I had continued to look, but I was unable to find an existing CMS that would do what I needed with multiple sites, so I determined that I needed to build my own. I chose CodeIgniter because it was lightweight and well structured.

So far, I have written thirteen modules for CoolBrew including products, store locator, Active Directory search, site administration, SEO tools, recipes, contests, job listings, lists, mailform, and pages. The system is running about eight sites at the moment.

Built with: LAMP, CodeIgniter, elements from hcgPublic

Responsible for Program Design, PHP Programming, Database Design, Default Template Design
